One-time setup needed

Connect to Firebase

This app needs a free Firebase database so students can sync in real time. Takes about 5 minutes — you only do this once.

1

Go to console.firebase.google.com and click Add project. Name it anything (e.g. "metabolism-session"). Skip Google Analytics.

2

In the left sidebar, click Build → Realtime Database → Create database. Choose "Start in test mode". Pick any region. Click Done.

3

You'll see a URL at the top of the page. It looks like:
https://metabolism-session-xxxxx-default-rtdb.firebaseio.com
Copy this URL.

4

Open this HTML file in a text editor (Notepad / VS Code). Find the line near the bottom that says const FB_URL = "https://fuel-metabolism-default-rtdb.asia-southeast1.firebasedatabase.app" and paste your URL in place of the placeholder. Save the file.

5

Go to netlify.com/drop — drag this HTML file into the browser window. You'll get a shareable link in seconds. Share it via Teams.

Default facilitator password: teach2024 — change the FAC_PASS variable in the file if you want a different one.
Note: Firebase test mode lasts 30 days — plenty of time for your session.
